## Banking Knowledge Assistant (In Progress)

### Overview

The Banking Knowledge Assistant is a Retrieval-Augmented Generation (RAG) system designed to allow users to upload banking policy documents and ask questions using natural language.

The goal is to improve access to organizational knowledge by combining document retrieval techniques with Large Language Models (LLMs).

### Problem Statement

Banks and financial institutions maintain large volumes of policy documents, operational procedures, compliance guidelines, and customer information resources.

Finding relevant information within these documents can be time-consuming and inefficient.

### Proposed Solution

The system will:

* Process PDF banking documents
* Extract and index document content
* Store embeddings in a vector database
* Retrieve relevant information based on user questions
* Generate context-aware responses using an LLM
